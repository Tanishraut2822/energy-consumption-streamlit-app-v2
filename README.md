# Smart Appliance Energy Consumption Predictor

This project predicts household appliance energy consumption using a Random Forest machine learning model.

## Features
- Predicts appliance energy usage (Wh)
- Provides energy-saving recommendations
- Built using Python, Pandas, Scikit-learn
- Deployed using Streamlit

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
